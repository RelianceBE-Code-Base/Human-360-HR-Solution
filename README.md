# Human-360-HR-Solution
Human 360 HR Solution is a comprehensive, end-to-end human resource management platform designed to streamline employee lifecycle management and optimize HR operations.
Built with modern web technologies, it provides a 360° view of employee data, enabling organizations to efficiently manage recruitment, onboarding, performance evaluations, leave tracking, payroll, compliance, and more — all in one centralized system.

Key Features

Employee 360° Profile – Consolidated view of personal, professional, and performance data.

Recruitment & Onboarding – Manage job postings, applications, and smooth onboarding workflows.

Performance Management – Set goals, track progress, and conduct appraisals with ease.

Leave & Attendance Tracking – Real-time leave requests, approvals, and attendance monitoring.

Payroll & Benefits – Automated payroll calculations and benefits management.

Compliance & Reporting – Maintain HR compliance and generate detailed analytics.

Tech Stack

Frontend: React / Typescript /SPFx

Backend: .NET 

Database: Azure SQL / SQL Server

APIs: REST / GraphAPI for seamless integrations.

Whether for small businesses or large enterprises, Human 360 HR Solution empowers HR teams to work smarter, make data-driven decisions, and improve employee experience.
